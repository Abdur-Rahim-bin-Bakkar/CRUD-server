# Simple CRUD API with Express & MongoDB

A simple REST API built with **Node.js**, **Express.js**, and **MongoDB** for basic CRUD operations.

This project allows you to:

- Create a user
- Read all users
- Read a single user
- Update a user
- Delete a user

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- CORS
